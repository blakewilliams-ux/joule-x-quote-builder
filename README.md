# joule-x-quote-builder

**JOULE X QUOTE TOOL**

## Main File

📄 **[index.html](index.html)** — The primary quote builder application. This is the main entry point for the Joule X Q2 2026 Quote Builder.

---

### Quick Start

Open `index.html` in your browser to launch the quote builder.

### Features

- Interactive quote configuration builder
- Real-time pricing calculations
- Support for multiple Joule X configurations (BBL, Halo, diVa, ClearSilk, ClearV, HaloTribrid, Touchup, Complete Resurfacing)
- Reference panel with configuration status
- Mobile-responsive design (PWA capable)

### Project Structure

- **index.html** — Main application file with complete HTML, CSS, and JavaScript
- **site-assets/** — Asset files (logos, configuration images)
- **manifest.webmanifest** — PWA manifest configuration
- **service-worker.js** — Service worker for offline support
